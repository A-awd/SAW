# GitHub Projects Standard

## Fields

- Area
- Repo
- Agent
- Risk
- Status
- Decision Required
- Due

## Status Flow

Inbox -> Planned -> In Progress -> Review -> Done

Blocked can be used from any status.

## Views

- AI OS
- Production Apps
- Commerce
- Automation
- High Risk
- Decisions Needed

## Rule

Cross-repo work belongs in GitHub Projects once Projects access is available.
